# MCreator & Ticker Datapack detector
A simple website that locally detects if a mod (or multiple mods) is an MCreator mod, or has a datapack function that runs every tick

# Usage
Simply go to https://ton185.github.io/mcreator-and-ticker-datapack-detector/

Once its open, you just need to drag and drop the jar file(s) that you want to check. I recommend using the defaults options unless you know what they do, for example if you disable the `Strict check` for tickers you will get some false positives.

If you want to have a local copy for whatever reason, you can [Download the zip](https://github.com/ton185/mcreator-and-ticker-datapack-detector/archive/refs/heads/main.zip) or `git clone` the repo, and then open `index.html` in your browser of choice.
